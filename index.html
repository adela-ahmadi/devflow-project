Git-native tracking

Sprint planning

Stop asking "is this done yet" in standup. When code merges, DevFlow reads the commit, matches it to a card, and moves it — automatically, in real time.

  • Respects branch naming and commit conventions
  • Full history, so nothing gets lost in transit
